# 🚗 Parking Automation System with QR (n8n)

Sistema automatizado de control de acceso para estacionamiento utilizando n8n, Telegram y Google Sheets.

## 🚀 Funcionalidades

* Generación de códigos QR
* Registro de accesos
* Automatización de entradas y salidas
* Almacenamiento en Google Sheets
* Interacción mediante bot de Telegram

## ⚙️ Tecnologías

* n8n
* Telegram Bot API
* Google Sheets
* JavaScript

## 🔄 Flujo

1. Usuario interactúa con el bot
2. Se genera un registro
3. Se asigna un identificador / QR
4. Se almacena en Google Sheets
5. Se valida acceso
